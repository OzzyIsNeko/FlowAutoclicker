## Build

Requirements:
- Visual Studio Build Tools with the C++ workload
- Windows SDK

```powershell
powershell -ExecutionPolicy Bypass -File .\build.ps1
```

The release executable is written to `build\Release\FlowAutoclicker.exe`.
